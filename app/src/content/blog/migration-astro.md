## Pourquoi ce changement ?

En tant qu'ingénieur DevOps, je voulais un site qui soit **rapide**, **sécurisé** et **automatisé**. 
J'ai découvert Astro...

### L'Architecture

Mon site est hébergé sur AWS S3, distribué par Cloudflare, et le tout est déployé via GitHub Actions.
